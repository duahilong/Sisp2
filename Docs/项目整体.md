# 项目名称 ：Sisp - System Installation Script Project

## 整体介绍
结合编写的工具、脚本和外部软件，来实现对多个或者单个硬盘的系统安装。通过对硬盘的初始化、格式化、分区、克隆操作系统等一系列操作，实现对硬盘的完整制作。


## 项目技术架构
项目整体所以使用Python编写，结合外部软件和Windows上的脚本，实现完整流程。
- 工具：ghost64.exe、bcdboot.exe 等，更详细的说明 [引用外部软件信息.md](引用外部软件信息.md)
- 脚本：硬盘分区脚本通过Windows上的 powershell 编写。
- 模块：通过python编写模块，来实现各个功能。

## 运行前置条件
- 项目运行环境为 Windows。
- 项目依赖 PowerShell 执行磁盘相关命令。
- 程序需以管理员权限运行。
- 外部软件 `ghost64.exe`、`bcdboot.exe` 需存在于配置指定的位置。
- 若未以管理员权限运行，程序应在流程开始前终止，并提示用户重新以管理员身份启动。

## 整体运行流程

``` 
运行软件
检查运行环境和管理员权限
读取json配置文件
获取硬盘信息
终端显示硬盘信息
用户选择硬盘（根据硬盘编号选择单个或多个硬盘）
初始化硬盘
验证初始化结果
硬盘分区和格式化
验证分区和格式化结果
调用外部软件ghost64.exe写入镜像
验证镜像写入结果
拷贝目录到特定位置
验证拷贝结果
调用外部软件bcdboot.exe创建引导记录
验证引导记录创建结果
验证整体完成并返回结果（通过/失败）
```
