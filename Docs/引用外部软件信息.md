# 引用软件外部信息

## Bcdboot.exe

### 存储位置
 - 项目中 `Sw` 文件夹内。
 - 相对路径：`Sw\bcdboot.exe`

#### 概述
BCDBoot 是一个命令行工具，用于配置 PC 或设备上的启动文件以运行 Windows 操作系统。主要应用场景：

1. 部署新 Windows 镜像后添加启动文件 — 在基于镜像的 Windows 部署中，用 BCDBoot 设置固件和系统分区以启动镜像。
2. 设置从 VHD（虚拟硬盘）启动 — 配置 PC 从包含 Windows 镜像的 VHD 文件启动。
3. 修复系统分区 — 当系统分区损坏时，用 Windows 分区中的新副本重建系统分区文件。
4. 设置或修复双启动菜单 — 在多系统 PC 上添加或修复启动菜单。


#### 命令行语法

```
BCDBOOT <source> [/l <locale>] [/s <volume-letter> [/f <firmware type>]] [/v] [/m [{OS Loader GUID}]] [/addlast or /p] [/d] [/c]
```
#### 关键要点
- BCDBoot 是 Windows 部署和启动修复的 核心工具 。
- 在 UEFI 系统上，它能直接操作 NVRAM 固件条目 。
- /s 参数用于 跨设备部署 （如 USB 或第二硬盘），此时不会修改当前设备的 NVRAM。
- /addlast 和 /p 互斥 ，分别控制启动顺序的末尾和保留原位。
- Windows 10+ 升级时默认 保留调试等特殊设置 ， /c 可强制清除。

## ghost64.exe

### 存储位置
 - 项目中 `Sw` 文件夹内。
 - 相对路径：`Sw\ghost64.exe`


#### 核心功能：

- 分区/磁盘备份 ：支持将分区或整个磁盘备份为 .gho 镜像文件
- 分区/磁盘克隆 ：直接克隆到另一分区或磁盘（Disk to Disk/Partition to Partition）
- 扇区级复制 ：逐扇区精确复制，保证数据完整性
- 增量/差异备份 ：支持后续增量备份以节省空间

#### 技术特性：

- 镜像格式：.gho（LZX压缩，最高65%压缩率）
- 加密支持：AES-256加密保护镜像文件
- 校验机制：SHA-1分段校验确保完整性
- 支持文件系统：NTFS、FAT32、exFAT

####  注意事项
1. 权限要求 ：需以管理员身份运行，或在WinPE环境中执行
2. 目标分区会被覆盖 ：使用 -sure 参数时务必确认目标正确
